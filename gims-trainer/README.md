# ГИМС-тренажер (офлайн, vanilla JS)

Локальный офлайн-тренажер подготовки к экзамену ГИМС без backend и без внешних библиотек.

## Что реализовано
- Четкое разделение режимов:
  - `Тренировка` (мгновенный разбор после каждого вопроса);
  - `Экзамен` (проверка только по завершении билета).
- Сценарии сессий:
  - `full` (тип + район),
  - `type-ticket`,
  - `area-ticket`,
  - `mistakes` (только вопросы с ошибками, без подмешивания).
- Обучение усилено мини-проверками:
  - каждый модуль считается завершенным только после мини-квиза.
- Адаптивная статистика:
  - attempts / correct / wrong / streakCorrect / lastAt / masteryScore.
- Аналитика:
  - слабые темы,
  - сильные темы,
  - топ проблемных вопросов,
  - расширенная история попыток.
- Экспорт/импорт прогресса в JSON.
- Fallback изображений в обучении:
  - при отсутствии файла показывается блок `Иллюстрация недоступна`.

## Архитектура файлов
- `index.html` — экраны и подключение модулей.
- `styles.css` — UI, responsive, focus-visible и accessibility-стили.
- `config.js` — все ключевые пороги/параметры.
- `data/modules.js` — учебные модули и мини-квизы.
- `data/questions.js` — тренировочный банк вопросов в единой схеме.
- `data/questions-archive.js` — архивные формулировки вопросов (неофициальный импорт из открытых источников).
- `core/storage.js` — localStorage, импорт/экспорт, очистка ошибок.
- `core/exam.js` — генерация билетов, вариативность, валидатор банка, аналитика.
- `ui/render.js` — отрисовка setup/learning/exam/result.
- `app.js` — оркестрация сценариев и пользовательских действий.

## Источники банка
- Базовый банк в проекте помечен как `training` (тренировочный).
- Архивный пакет помечен как `archive:*` и добавляет исторические формулировки из открытых онлайн-банков.
- Важно: архивный пакет не следует считать официальной публикацией МЧС, он предназначен для расширения вариативности тренировки.
- Для подключения подтвержденного официального банка можно добавить/заменить файл,
  который выставляет `window.QuestionBankOfficialData`.

## Запуск
1. Откройте `index.html` в браузере.
2. Интернет не требуется.
